## 四、各模块技术规划

### 1. 爬虫模块
- **目标页面：** 豆瓣电影各影片短评页（需登录 Cookie）
- **核心字段：** 见上方字段表
- **翻页逻辑：** 每页 20 条，通过 `start` 参数翻页（start=0, 20, 40, ...）
- **URL 模板（以《飞驰人生3》好评为例）：**
  ```
  https://movie.douban.com/subject/37311135/comments?percent_type=h&start={start}&limit=20&status=P&sort=new_score
  ```
- **三部影片 Subject ID：**
  | 影片 | Subject ID | 好评 URL 参数 |
  |---|---|---|
  | 《飞驰人生3》 | 37311135 | percent_type=h |
  | 《镖人：风起大漠》 | 36474027 | percent_type=h |
  | 《惊蛰无声》 | 37242440 | percent_type=h |
- **必须携带 Cookie：** 豆瓣超过第1页需登录验证，headers 中必须加 Cookie
- **获取 Cookie 方法：** 浏览器登录豆瓣 → F12 → Network → 找 comments 请求 → Request Headers → 复制 Cookie 字段
- **注意事项：**
  - 添加请求头（User-Agent）与随机延迟（2—4秒），避免被封
  - 按页爬取，记录 `source_page`
  - 原始数据存为 CSV，编码 UTF-8-BOM（兼容 Excel）
  - 每部影片用 `movie_name` 字段区分，最终合并为一个 CSV

### 2. 数据清洗与预处理（在聚类/情感分析代码中实现）
- 依据 `id` 或"影片名+文本+时间"去重
- 删除评论文本为空、评分或时间缺失的记录
- 过滤极短无效文本（如纯表情、单字）
- 正则清除特殊符号、表情包、冗余空格
- 使用 `jieba` 分词，构建电影场景自定义词典
- 去除停用词，构造辅助变量（评论字数、评论日期等）

### 3. 文本聚类模块
- **方法：** TF-IDF 提取关键词权重 + K-Means 聚类（或 LDA 主题模型）
- **对象：** 三部影片评论分别建库，也可合并后加 `movie_name` 标签对比
- **可视化：** 词云图、聚类主题关键词柱状图、降维散点图（t-SNE/PCA）
- **人工命名：** 根据高频词与代表性评论对主题命名
- **预期主题示例：**
  - 《飞驰人生3》：喜剧笑点 / 续作情怀 / 赛车燃点
  - 《镖人：风起大漠》：动作设计 / 武侠氛围 / 改编还原度
  - 《惊蛰无声》：悬疑设定 / 节奏控制 / 角色塑造

### 4. 情感分析模块
- **方法：** 基于情感词典（如 SnowNLP / VADER / 自定义词典）对每条评论打分
- **输出：** 情感得分标准化至 0—1 区间
- **对照分析：** 情感得分 vs. 用户评分（rating）相关性
- **可视化：** 三部影片情感得分分布对比图、情感-评分散点图

### 5. 探索性分析模块（EDA）
包含以下三个子分析方向：

#### 5.1 口碑分布对比
- 评分分布直方图 / 箱型图（三部影片并排对比）
- 情感得分分布对比
- 评论字数分布对比

#### 5.2 口碑时间演化分析
- 按 `created_at` 日期聚合，绘制：
  - 每日平均评分趋势
  - 每日平均情感得分趋势
  - 每日评论数量趋势
- 观察上映初期 vs. 假期后段的口碑变化规律

#### 5.3 高赞评论规律分析
- 以 `votes_count` 为核心，划分普通评论 vs. 高赞评论
- 对比两类评论的：
  - 情感倾向差异
  - 评论长度差异
  - 关键词类型差异
- 探究何种评论更容易获得平台用户认可
