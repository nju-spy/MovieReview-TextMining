# 探索性数据分析 (EDA) 模块

本模块对豆瓣短评数据进行多维度探索性分析，包含四个分析脚本，共输出 11 张可视化图表。

---

## 文件结构

| 文件 | 说明 | 输出图表 |
|------|------|----------|
| `rating 与 发布时间段的关系.py` | 分析不同时段的评分分布特征 | 图1-2 |
| `口碑分布对比.py` | 三部影片的评分/字数/高赞占比对比 | 图3-5 |
| `口碑时间演化分析.py` | 每日评论量、评分、情感比例趋势 | 图6-8 |
| `高赞评论规律分析.py` | 高赞 vs 普通评论的差异分析 | 图9-11 |

## 各脚本详解

### 1. rating 与 发布时间段的关系.py

**输入**：`../1.爬虫/合并数据.csv`

**分析内容**：
- 不同时段（上午/下午/晚上/深夜）的星级分布比例
- 不同星级用户的发布时段偏好

**输出图表**：
- **图1**：时段→星级分布（堆叠条形图）
- **图2**：星级→时段分布（堆叠条形图）

**核心发现**：
- 约 80% 评论集中在下午和晚上（12:00-24:00）
- 深夜差评比例最高，好评比例最低
- 上午时段好评比例相对最高

---

### 2. 口碑分布对比.py

**输入**：`../2.预处理/*_cleaned.csv`

**分析内容**：
- 三部影片评分分布对比
- 评论字数分布对比（反映表达意愿）
- 高赞评论占比对比（反映社区活跃度）

**输出图表**：
- **图3**：评分分布对比（分组柱状图）
- **图4**：字数分布对比（箱线图）
- **图5**：高赞占比对比（柱状图）

**核心指标**：
- 各影片平均评分、中位数评分
- 平均字数、中位数字数
- 高赞评论占比百分比

---

### 3. 口碑时间演化分析.py

**输入**：`../2.预处理/*_cleaned.csv`

**分析内容**：
- 每日评论数量趋势（热度演化）
- 每日平均评分趋势（口碑演化）
- 每日好/中/差评比例趋势（情感演化）

**输出图表**：
- **图6**：每日评论数量趋势（三线折线图）
- **图7**：每日平均评分趋势（三线折线图）
- **图8**：每日情感比例趋势（堆叠面积图）

**核心发现**：
- 评论量峰值日期（通常为上映前1-2天）
- 评分波动范围（观察"高开低走"或"逆袭"现象）
- 情感比例演化（假期后期是否出现审美疲劳）

---

### 4. 高赞评论规律分析.py

**输入**：`../2.预处理/*_cleaned.csv`

**分析内容**：
- 高赞评论 vs 普通评论的评分分布差异
- 高赞评论 vs 普通评论的字数分布差异
- 高赞评论 vs 普通评论的情感倾向差异

**统计检验**：
- Mann-Whitney U 检验（非参数检验，比较两组独立样本）
- 检验评分差异显著性
- 检验字数差异显著性

**输出图表**：
- **图9**：高赞评分分布（箱线图）
- **图10**：高赞字数分布（箱线图）
- **图11**：高赞情感倾向对比（分组柱状图）

**核心问题**：
1. 高赞评论是否显著更长？→ 长文通常更有信息价值
2. 高赞评论评分是否更极端？→ 极端评价更容易引发共鸣
3. 哪类情感倾向更容易获赞？→ 正面评价 vs 负面评价的传播力

## 使用说明

1. 确保预处理后的 CSV 文件已生成在 `../2.预处理/` 目录
2. 直接运行各 Python 脚本：
   ```bash
   python "rating 与 发布时间段的关系.py"
   python 口碑分布对比.py
   python 口碑时间演化分析.py
   python 高赞评论规律分析.py
   ```
3. 所有图表将自动保存到当前目录（`5.EDA/`）
4. 统计结果将输出到控制台
