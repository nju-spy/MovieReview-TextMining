# 电影评论数据预处理项目

对电影评论 CSV 数据进行批量清洗与预处理，主要面向后续的文本分析、情感分析、词频统计、主题建模或可视化分析等任务。

对多部电影分别配置输入文件、输出文件和电影专属词典，并统一完成评论去重、字段过滤、文本清洗、中文分词、停用词过滤、时间字段转换和衍生字段构造。

## 项目功能

本项目的预处理流程主要包括：

1. 读取每部电影对应的原始评论 CSV 文件。
2. 根据评论文本和发布时间进行去重。
3. 删除缺失或无效评论，包括：
   - 评论文本为空；
   - 评分为空；
   - 发布时间为空；
   - 评论长度过短。
4. 使用正则表达式清洗评论文本，去除：
   - emoji；
   - HTML 标签；
   - 非中文、英文、数字和常用中文标点的特殊字符；
   - 多余空白字符。
5. 使用 `jieba` 进行中文分词。
6. 加入每部电影的专属词典，提高人物名、片名、专业词汇等分词准确率。
7. 使用全局停用词表过滤无意义词语。
8. 构造衍生字段：
   - 评论日期；
   - 评论发布时间段；
   - 是否为高点赞评论。
9. 导出清洗后的 CSV 文件。


## 输入数据

| 字段名 | 说明 |
|---|---|
| `movie_name` | 电影名称 |
| `comment_type` | 评论类型 |
| `comment_text` | 评论正文 |
| `rating` | 用户评分 |
| `votes_count` | 评论点赞数 |
| `created_at` | 评论发布时间 |
| `comment_length` | 评论长度 |
| `source_page` | 来源页码或来源信息 |

## 输出字段说明

清洗后的 CSV 文件包含以下字段：

| 字段名 | 说明 |
|---|---|
| `movie_name` | 电影名称 |
| `comment_type` | 评论类型 |
| `comment_text` | 清洗后的评论文本 |
| `tokens` | 分词并去停用词后的结果 |
| `rating` | 用户评分 |
| `votes_count` | 评论点赞数 |
| `is_high_vote` | 是否为高点赞评论，1 表示是，0 表示否 |
| `created_at` | 评论发布时间 |
| `comment_date` | 评论日期 |
| `time_period` | 评论发布时间段 |
| `comment_length` | 清洗后的评论长度 |
| `source_page` | 来源页码或来源信息 |

## 时间段划分规则

脚本会根据 `created_at` 的小时部分生成 `time_period` 字段。

| 时间范围 | 标记 |
|---|---|
| 06:00 - 11:59 | 上午 |
| 12:00 - 17:59 | 下午 |
| 18:00 - 23:59 | 晚上 |
| 00:00 - 05:59 | 深夜 |
| 无法解析 | 未知 |

## 数据处理流程

整体处理逻辑如下：

```text
读取原始 CSV
    ↓
添加电影专属词典
    ↓
去重与基础字段过滤
    ↓
正则清洗评论文本
    ↓
重新计算评论长度
    ↓
过滤过短评论
    ↓
jieba 中文分词
    ↓
停用词过滤
    ↓
构造日期、时段、高点赞标记
    ↓
保留指定字段
    ↓
导出清洗后的 CSV
```
